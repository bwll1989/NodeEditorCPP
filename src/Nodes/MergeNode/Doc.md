# MergeNode 使用说明

## 用途
MergeNode 用于把多路输入（VariableData）里的某些字段“挑出来合并”为一个新的 VariableData 输出。常用于把不同节点输出的数据拼成一份统一的数据包，供下游脚本/网络/控制节点使用。

## 端口
- 输入（默认 5 个，可编辑）
  - INPUT 0..N：VariableData
- 输出（1）
  - OUTPUT 0：VariableData（合并后的结果）

## 节点界面
节点内部是一张映射表（可添加/删除行），每一行表示“从哪个输入端口取哪个字段，输出到哪个字段”：
- ID：输入端口索引（例如 0 表示 INPUT 0）
- Name：要读取的字段名（从该输入的 VariableData 中取值）
- Rename：输出字段名（可为空；为空时输出字段名=Name）

## 使用步骤
1. 连接若干数据源到 INPUT 端口（例如 JSON/脚本/设备节点的输出）。
2. 在表格里添加若干行，填写 ID/Name/Rename。
3. 从 OUTPUT 0 得到合并后的 VariableData。

## 注意事项
- 同一个输出字段名如果被多行写入，后面的会覆盖前面的。
- 如果某个 ID 对应的输入端口没有连接或没有数据，该行不会产出有效值。
