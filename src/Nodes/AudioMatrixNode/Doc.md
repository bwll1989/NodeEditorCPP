# Audio Matrix 节点

## 1. 节点说明

8×8 音频路由矩阵：将多路输入按界面矩阵分配到多路输出，适合多轨混音与监听切换。

## 2. 端口说明

### 输入

- **IN1**～**IN8**（AudioData）：8 路音频输入。

### 输出

- **OUT1**～**OUT8**（AudioData）：8 路音频输出。

## 3. 界面说明

8×8 电平矩阵：点击交叉点可将对应输入送到对应输出（增益/开关依实现）。

## 4. 使用说明

1. 将各音源接到 IN1～IN8。
2. 在矩阵中勾选路由关系。
3. 将 OUTn 接到声卡或下级处理。

## 5. 示例

4 路播放器 + 2 路话筒 → Audio Matrix → OUT1/2 主 LR，OUT3 耳返。
