# AudioMatrixNode 使用说明

## 用途
AudioMatrixNode 用于把多路输入音频按“矩阵”方式路由/混合到多路输出，常用于：
- 多路音源混音
- 声道重分配（某一路输入送到多个输出）
- 输出编组（多路输入汇到一路输出）

## 输入端口（AudioData）
- 多路输入（通常为 6 路，按节点配置为准）。

## 输出端口（AudioData）
- 多路输出（通常为 8 路，按节点配置为准）。

## 界面操作
- 矩阵面板：每个格子代表“输入 i → 输出 j”的混合量。
- 一般用法：
  - 把不需要的格子调到最小（静音）。
  - 只保留你希望的路由/混合路径。

## 快速上手
1. 把音频源分别接到输入端口。
2. 把需要的输出端口接到后续处理或音频输出设备节点。
3. 在矩阵面板里把需要的路径打开/调大（例如：In1 → Out1）。
4. 如果输出过大产生失真，降低相关路径的混合量。

## 注意事项
- 多路叠加容易过载失真，建议逐步加大混合量并观察听感。
- 若某路输出没有声音，先检查矩阵对应路径是否为静音。
