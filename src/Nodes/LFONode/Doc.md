# LFO 节点

## 1. 节点说明

LFO（低频振荡器）按设定波形、频率和振幅**持续输出**变化的数值，常用于自动化缓变、呼吸灯、周期性调制等。通过内部定时器按采样率生成波形；输入端口可用来**启动**振荡（收到布尔 `true` 时开始运行）。

支持三种波形：正弦（Sine）、方波（Square）、三角波（Triangle）。

## 2. 端口说明

### 输入

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0 | （默认） | VariableData | 值为布尔 `true` 时启动振荡（`running = true`） |

### 输出

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0 | （默认） | VariableData | 当前波形采样值；附带 `method` 字段（波形类型枚举） |

## 3. 界面说明

节点**无内嵌操作面板**（波形与参数通过外部控制或属性系统设置）。可参考的参数含义：

| 参数 | 含义 | 典型范围（参考） |
|------|------|------------------|
| method | 波形：0=正弦，1=方波，2=三角 | — |
| frequency | 频率（Hz） | 默认约 5 |
| amplitude | 振幅 | 默认约 10 |
| phase | 相位 | 默认 0 |
| sampleRate | 采样率（次/秒），决定刷新间隔 | 默认约 10 |
| running | 是否正在输出波形 | true / false |

**外部控制地址**（相对节点地址，经全局事件总线）：

- `/method` — 切换波形（整型）
- `/frequency`、`/amplitude`、`/phase`、`/sampleRate` — 数值参数
- `/running` — 启停振荡（布尔）

## 4. 使用说明

1. 将 LFO 输出接到需要调制强度的下游节点。
2. 通过外部控制或属性设置频率、振幅、波形与采样率。
3. 向输入发送 `true` 或设置 `/running` 为 true 开始输出；设为 false 停止定时器。
4. 修改波形或主要参数时，内部时间会重置，避免相位跳变混乱。

采样率过低时波形粗糙；为 0 或极小时输出恒为 0。

## 5. 示例

**呼吸灯：** 正弦波、频率 0.2 Hz、振幅 1，输出接调光节点。演出开始时向输入发送 `true`。  
**遥控改速：** 将 `/frequency` 绑定到控台旋钮，现场调整闪烁快慢而无需改图。
