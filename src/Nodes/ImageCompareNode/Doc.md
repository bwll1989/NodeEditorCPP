# ImageCompareNode 使用说明

## 用途
ImageCompareNode 用于比较两张图像的相似度/差异度，并输出一个数值结果。适合做画面变化检测、画面对齐检查、自动挑选相似帧等。

## 端口
- 输入（2）
  - IMAGE 0（端口 0）：ImageData
  - IMAGE 1（端口 1）：ImageData
- 输出（1）
  - OUTPUT 0（端口 0）：VariableData（数值结果）

## 方法选择
节点界面可选择对比方法（methodIndex）：
- 0：MSE（均方误差，范围约 0~1，越小越相似）
- 1：SSIM（结构相似度，范围约 0~100，越大越相似）
- 2：PSNR（峰值信噪比，单位 dB，越大越相似）

说明：如果界面里出现了其他方法选项，可能未在当前版本中实现或结果不稳定，建议先用上述 0-2 三种方法。

## 使用步骤
1. 将两路图像分别连接到 IMAGE 0 / IMAGE 1。
2. 在节点界面选择对比方法。
3. 从 OUTPUT 0 读取结果数值，并在下游做阈值判断或显示。

## 外部控制（可选）
- `/method`：设置方法索引（int）

## 注意事项
- 节点会自动把两张图像转换为灰度并统一到共同的最小尺寸后再计算。
- 任意一路输入为空时不会更新结果。
