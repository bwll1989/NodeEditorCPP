# ImageConstNode 使用说明

## 用途
ImageConstNode 用于生成一张指定尺寸、指定颜色的“纯色图像”，并从输出端口输出，常用于测试画面、占位图、遮罩底色等。

## 端口
- 输入（3）
  - WIDTH（端口 0）：宽度（int）
  - HEIGHT（端口 1）：高度（int）
  - RED（端口 2）：颜色（string）

说明：
- 当前版本端口 2 的名字显示为 RED，但实际用于传入“颜色字符串”，不是单独的红色分量。
- 颜色字符串支持 `#AARRGGBB`（推荐），例如：`#FFFF0000`（不透明红色）、`#800000FF`（半透明蓝色）。

- 输出（1）
  - Image（端口 0）：ImageData，纯色图像

## 使用步骤
1. 在节点界面中设置宽度/高度与颜色，或用端口输入动态设置 WIDTH/HEIGHT/颜色字符串。
2. 将输出 Image 连接到下游图像处理/显示/录制节点。

## 注意事项
- WIDTH/HEIGHT 小于等于 0 时会被自动修正为 1。
- 输入为空时不会更新输出。
