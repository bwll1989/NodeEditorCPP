# Image Constant 插件

本插件包含三种节点：**Image Constant**（纯色图）、**Text To Image**（文字图）、**Image Layout**（多图拼接）。

---

## Image Constant

### 1. 节点说明

生成指定尺寸、指定颜色的纯色图像，适合占位、蒙版或测试画面。

### 2. 端口说明

#### 输入

- **WIDTH**（VariableData）：宽度（像素）。
- **HEIGHT**（VariableData）：高度。
- **RED**（VariableData）：颜色（字符串或 QColor 形式），会更新颜色编辑器。

#### 输出

- **Image**（ImageData）：生成的 RGBA 图像。

### 3. 界面说明

- 宽度、高度数值框。
- 颜色预览区与「颜色编辑器」按钮（可弹出独立颜色窗口）。

### 4. 使用说明

在界面中设置宽高和颜色，或通过输入端口动态覆盖；修改后自动重新生成图像。

### 5. 示例

Image Constant（1920×1080 黑色）→ 作为 Image Layout 的背景层。

---

## Text To Image

### 1. 节点说明

将文字渲染为图像，支持字体、字号、颜色、对齐等参数。

### 2. 端口说明

#### 输入（均为 VariableData，可选）

- **WIDTH** / **HEIGHT**：画布尺寸。
- **TEXT**：文字内容。
- **FONT** / **FONTSIZE**：字体与字号。
- **TEXT_COLOR** / **BG_COLOR**：文字色与背景色。
- **HALIGN** / **VALIGN**：水平、垂直对齐（索引）。

#### 输出

- **Image**（ImageData）：渲染结果。

### 3. 界面说明

文本框、字体选择、字号、粗体/斜体/下划线、对齐、边距、换行，以及文字色/背景色按钮。

### 4. 使用说明

在节点内编辑文字样式，或用端口传入动态文案与尺寸。

### 5. 示例

String 变量 → TEXT 端口 → Text To Image → 叠加到视频画面上。

---

## Image Layout

### 1. 节点说明

将多路输入图像按网格等方式拼成一张图（默认 4 路输入，可增删端口）。

### 2. 端口说明

#### 输入

- **Image1**～**ImageN**（ImageData）：待拼接的图像，端口数量可编辑。

#### 输出

- **Image**（ImageData）：拼接后的图像。

### 3. 界面说明

画布宽高、布局方式、行列数、间距、背景色及颜色编辑器。

### 4. 使用说明

1. 按需增加输入端口并连接各图像源。
2. 选择布局（如网格）并设置行列、间距。
3. 从 Image 输出口取合成结果。

### 5. 示例

四路摄像头 → Image Layout（2×2 网格）→ Spout Out。
