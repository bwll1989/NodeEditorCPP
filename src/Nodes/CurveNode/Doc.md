# CurveNode 使用说明

## 用途
CurveNode 用于播放时间轴曲线：按时间输出曲线值（X/Y）与播放状态，常用于做参数随时间变化、灯光/视频/特效的自动化控制。

## 端口
- 输入（1）
  - TRIGGER：布尔触发（true/false），用于控制播放/暂停（可选）
- 输出（4）
  - TIME：当前时间（秒）
  - VALUE_X：当前曲线 X 值
  - VALUE_Y：当前曲线 Y 值
  - STATUS：当前是否在播放（true/false）

## 节点界面
- Start：开始/暂停播放
- Loop：是否循环播放
- 时间轴编辑器：编辑关键帧与曲线（可弹出独立窗口编辑）

## 使用步骤
1. 打开编辑器，添加关键帧并调整曲线。
2. 点击 Start 开始播放；需要循环时勾选 Loop。
3. 将 VALUE_X / VALUE_Y 连接到下游节点参数输入；用 TIME / STATUS 做同步或条件触发。
4. 如需外部触发，可给 TRIGGER 输入 true/false 控制播放/暂停。

## 外部控制（可选）
- `/start`：播放/暂停（bool）
- `/loop`：循环开关（bool）

## 常见问题
- 没有输出：确认已开始播放，且曲线里有关键帧。
- Loop 不生效：确认 Loop 已勾选，且外部控制未覆盖本地设置。
