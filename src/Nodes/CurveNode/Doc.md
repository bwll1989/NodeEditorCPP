# Curve 节点

## 1. 节点说明

Curve 提供**可编辑时间轴曲线**播放能力：在时间轴上绘制关键帧曲线，播放时按当前时间输出曲线上的数值，适合自动化参数、演出编创、与进度同步的控台逻辑。

支持循环播放；可通过 TRIGGER 输入或外部控制启停。

## 2. 端口说明

### 输入

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0 | TRIGGER | VariableData | 布尔值：`true` 开始播放，`false` 暂停并回到起点 |

### 输出

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0 | PROGRESS | VariableData | 时间轴上的进度相关数值（由编辑器回传） |
| 1、2… | Curve1、Curve2… | VariableData | 各条曲线在当前播放位置的 Y 值（按曲线数量扩展） |

输出端口数量**可编辑**（至少包含 PROGRESS 与若干曲线端口）。

## 3. 界面说明

- **时间显示**：大号只读时间（`hh:mm:ss:zzz`），显示当前播放头位置。
- **启动 / 停止**：可切换按钮，控制播放；播放中显示「停止」。
- **循环**：勾选后时间轴循环播放。
- **编辑曲线**：打开独立「曲线编辑器」窗口，在内嵌 Web 时间轴中编辑关键帧；关闭窗口后编辑内容仍保存在节点内。

**外部控制地址：**

- `/start` — 播放/暂停（布尔，绑定 `start` 属性）
- `/loop` — 是否循环（布尔）

## 4. 使用说明

1. 点击「编辑曲线」绘制时间轴与曲线。
2. 将 TRIGGER 或 `/start` 用于开始/停止播放。
3. 将 Curve1、Curve2 等输出接到需要被曲线驱动的参数（调光、位置、切换等）。
4. 需要循环时勾选「循环」或设置 `/loop` 为 true。

时间轴数据随工程保存（`values.timeline` 或 `timelineString`）；循环状态保存在 `values.loop`。

## 5. 示例

**演出自动化：** 一条曲线控制主亮度（Curve1），另一条控制色温（Curve2）；演出开始时 TRIGGER 置 true。循环勾选后可做待机呼吸效果。  
**外部控台同步：** 将 `/start` 映射到 MIDI 或 OSC，由控台统一启停多条 Curve 节点。
