# VST3PluginNode 使用说明

## 用途
在音频链路中加载并运行一个 VST3 插件（效果器/处理器）。当前为 alpha 版本，插件兼容性不保证。

说明：
- 程序会扫描 VST3 插件目录，并为每个插件注册一个独立的节点类型（节点名通常为插件文件名）。

## 插件目录
优先使用可执行程序目录下：
- `<程序目录>/plugins/VST3`

若无法创建目录，会回退到用户文档目录下：
- `<文档>/plugins/VST3`

将 `.vst3` 放入目录后，重启程序即可看到对应节点。

## 端口
端口类型均为 AudioData（共享音频缓冲区）。

### 输入（动态）
输入端口数量由插件实际输入总线/通道决定：
- 单声道插件：通常为 1 路输入
- 立体声插件：通常为 2 路输入（L/R）

### 输出（动态）
输出端口数量由插件实际输出总线/通道决定：
- 单声道插件：通常为 1 路输出
- 立体声插件：通常为 2 路输出（L/R）

## 参数/界面
- Controller：打开/关闭插件的原生编辑界面（插件支持时可用）

## 使用步骤
1. 把插件放到插件目录并重启程序，添加对应的 VST3 节点。
2. 将上游音频输出连接到本节点输入端口（按通道对应连接）。
3. 将本节点输出端口连接到下游音频处理/输出节点。
4. 需要调参时点击 Controller 打开插件界面。

## 注意事项
- 插件需要支持 32 位浮点音频处理；不支持时可能无法工作。
- 节点的输入/输出端口数量可能不相等（由插件决定），连接时请按通道含义对应。
