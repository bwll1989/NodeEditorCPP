# VST3 Plugin 节点

## 1. 节点说明

加载 VST3 插件处理音频：输入/输出端口数量与通道数由插件决定。支持打开插件原生界面调节参数。

## 2. 端口说明

### 输入

- 若干 **AudioData** 输入端口（数量随插件输入总线变化）。

### 输出

- 若干 **AudioData** 输出端口（数量随插件输出总线变化）。

具体端口名在节点上按插件总线显示。

## 3. 界面说明

- 插件路径/选择（创建节点时指定）。
- **打开界面** 按钮：显示 VST3 编辑器窗口。

## 4. 使用说明

1. 添加节点并选择 VST3 插件文件。
2. 将上游 AudioData 接到插件输入。
3. 输出接到声卡或下一效果器；需要时用插件 UI 调参。

## 5. 示例

Audio Device In → VST3（混响）→ VST3（压缩）→ Audio Device Out，人声效果链。
