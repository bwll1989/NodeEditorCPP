# USR_IO808Node

## 1. 节点说明

通过 **Modbus TCP** 控制有人 USR-IO808 等 8 路开关量模块：写入数字输出（DO），读取数字输入（DI）。连接后定时轮询，并支持写命令队列。

## 2. 端口说明

### 输入

| 端口 | 类型 | 说明 |
|------|------|------|
| DO 0～7 | VariableData | 布尔值，设置对应数字输出通断 |

### 输出

| 端口 | 类型 | 说明 |
|------|------|------|
| DI 0～7 | VariableData | 布尔值，对应当前数字输入状态 |

## 3. 界面说明

- **主机 / 端口 / 从站 ID**：Modbus TCP 连接参数（默认端口代码中为 8080，常见设备为 502）。
- **连接状态**：TCP 是否就绪。
- **DO 0～7**：复选框控制输出。
- **DI 0～7**：只读显示输入。
- **全部读取**：手动刷新 DI/DO 状态。

外部控制：`/host`、`/port`、`/serverId`、`/DO0`～`/DO7`。

## 4. 使用说明

1. 填写模块 IP、端口、从站 ID，连接成功后 DO 可操作。
2. 勾选 DO 或通过输入端口写入 `true`/`false`。
3. 从 DI 输出端口读取传感器或按钮状态。
4. 连接后约每秒自动轮询一次；写操作经队列依次下发。

## 5. 示例

逻辑节点根据 DI 3 为高 → 向 DO 1 写入 `true` 驱动继电器；DI 3 接条件，DO 1 接本节点输入。
