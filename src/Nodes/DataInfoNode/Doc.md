# DataInfoNode 使用说明

## 用途
DataInfoNode 用于把输入的 VariableData（通常是一个键值表/嵌套结构）以树形方式展示，便于在流程中快速查看数据内容与层级；同时将输入数据原样透传到输出端口，方便你把它插在任意位置做“旁路监控”。

## 端口
- 输入（1）
  - INPUT 0：VariableData
- 输出（1）
  - OUTPUT 0：VariableData（透传）

## 使用步骤
1. 将需要查看的数据源（例如 JSON/脚本/网络输入等）连接到 INPUT 0。
2. 在 DataInfoNode 的界面里展开树形结构，查看键、值与层级。
3. 如需不中断数据流，把 OUTPUT 0 继续接到下游节点。

## 注意事项
- 输入为空时不会更新显示。
- 数据量很大时，展示可能会变慢；建议只在调试时使用，或只在关键节点处插入。
