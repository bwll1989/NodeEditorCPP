# Switch 节点

## 1. 节点说明

Switch 根据**索引**从多路输入中选择一路，转发到单一输出。类似多路开关：INDEX 决定当前选中的是 INPUT 0、INPUT 1 还是其它数据端口（最后一个输入端口固定为 INDEX）。

## 2. 端口说明

### 输入

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0 … N-2 | INPUT 0、INPUT 1 … | VariableData | 候选数据源 |
| 最后一个 | INDEX | VariableData | 整型索引，选择输出 `INPUT[index]` 的数据 |

默认共 3 个输入（INPUT 0、INPUT 1、INDEX）；**可编辑**以增加更多 INPUT，INDEX 始终为最后一个端口。

### 输出

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0 | OUTPUT 0 | VariableData | 当前 INDEX 选中的那路输入数据 |

## 3. 界面说明

无内嵌面板。当前索引可通过节点属性或外部控制设置。

**外部控制地址：**

- `/index` — 设置当前选中的输入索引（整型）

## 4. 使用说明

1. 将多路备选数据接到 INPUT 0、INPUT 1…
2. 将索引源（数值或来自逻辑节点）接到 INDEX。
3. 从 OUTPUT 0 取当前选中通道的数据。
4. INDEX 或任一路 INPUT 变化时，输出会更新。

索引保存在工程 `values.val` 中。

## 5. 示例

**三路视频源切换：** INPUT 0/1/2 接三个源，INDEX 接控台页码或 Logic 比较结果；输出接显示链路。  
**远程切路：** 将 `/index` 绑定 OSC，控台发送 0/1/2 即可切换。
