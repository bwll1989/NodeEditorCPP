# Count 节点

## 1. 节点说明

Count 在每次输入更新时，用 **JavaScript 条件表达式**判断是否「命中」；若表达式为真，内部计数器加 1，并输出当前计数值。适合统计满足条件的次数（如某状态出现次数、达标帧数等）。

支持外部清零与读取/设置计数值。

## 2. 端口说明

### 输入

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0 | （默认） | VariableData | 每次更新时用表达式求值；为真则计数 +1 |

输入端口数量**可编辑**（逻辑仍使用同一输入数据）。

### 输出

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0 | （默认） | VariableData | 当前计数值（整数） |

## 3. 界面说明

- **计数**：显示当前累计值（可拖动调整，并与外部 `/count` 同步）。
- **条件**：JS 表达式编辑框；输入数据在表达式中为全局变量 `$input`（与输入的键值映射一致）。默认示例：`$input["default"]`。
- **clear**：将计数清零。

**外部控制地址：**

- `/clear` — 发送 true 时清零（与 clear 按钮相同）
- `/count` — 读取或设置当前计数值

## 4. 使用说明

1. 在「条件」中编写返回布尔值的 JS，例如：`$input["value"] > 100`。
2. 将需要统计的数据源接到输入。
3. 输出接显示、比较或分支节点。
4. 场次切换时点击 clear 或发送 `/clear`。

表达式错误时保持原计数并输出当前值。

配置保存在 `values.expression`。

## 5. 示例

**统计超标次数：** 音频分析结果接入输入，条件 `$input["level"] > 0.8`，输出接 Data Info 或阈值报警。  
**控台复位：** 将 `/clear` 绑定复位按钮，一键清零。
