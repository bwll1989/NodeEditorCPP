# HotKey 节点

## 1. 节点说明

HotKey 在系统中注册最多 **5 组**全局快捷键。操作员按下已启用且配置好的组合键时，对应输出端口会更新数据，用于手动触发、计数反馈或布尔状态切换。

无输入端口，适合作为流程的「人工触发源」。

## 2. 端口说明

### 输入

无。

### 输出

| 端口 | 名称 | 数据类型 | 说明 |
|------|------|----------|------|
| 0–4 | （对应第 1–5 组热键） | VariableData | 主值为布尔（按钮当前绿/红状态）；附带 `count` 字段为按下累计次数 |

## 3. 界面说明

每组热键一行，包含：

- **HotKey（启用）**：勾选后该组快捷键才生效。
- **按键编辑框**：设置组合键（如 `Ctrl+1`）。
- **状态按钮**：显示按下次数；每次按下会在绿/红之间切换，并递增计数。
- **reset**：将该组计数清零，状态复位为未按下（红）。

配置会随工程保存（`hotkey1`…`hotkey5`，含 `enable` 与 `value`）。

## 4. 使用说明

1. 勾选要使用的行，在编辑框中录制快捷键。
2. 将 OUTPUT 0–4 接到下游（如 Inject 的 TRIGGER、Condition 等）。
3. 演出中按下快捷键即可触发对应分支。
4. 需要「纯计数」时读取输出中的 `count` 字段；需要「开关状态」时读取主布尔值。

未启用的热键不会注册，避免误触。

## 5. 示例

**导播手动切场景：** 5 组热键分别映射场景 1–5，输出接 Inject 的不同 TRIGGER 端口。  
**调试计数：** 输出 `count` 接显示或日志，reset 在每场次开始前清零。
