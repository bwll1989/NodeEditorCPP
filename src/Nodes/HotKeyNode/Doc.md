# HotKeyNode 使用说明

## 用途
HotKeyNode 用于监听系统热键，并把每次触发转换为输出信号。适合做键盘控制、现场操作快捷触发、调试触发等。

## 端口
- 输入：无
- 输出（5）
  - OUTPUT 0..4：VariableData

每个输出包含：
- `default`：当前热键状态（bool）
- `count`：触发次数（int）

## 节点界面
节点提供 5 组热键配置，每组包含：
- Enable：启用/禁用该热键
- 快捷键编辑器：设置组合键
- 状态/计数显示与重置

## 使用步骤
1. 在每一组里设置快捷键组合，并勾选 Enable。
2. 将对应的 OUTPUT 端口连接到需要触发的下游节点。
3. 按下热键后，该输出端口会更新（count 增加），用于触发下游逻辑。

## 注意事项
- 避免与系统/其他软件快捷键冲突。
- 只有启用状态的热键会响应。
