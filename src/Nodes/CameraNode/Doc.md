# Camera 节点

## 1. 节点说明

从本地摄像头采集实时画面并输出图像。在独立线程中采集，减轻界面卡顿。切换设备时会自动重启采集。

## 2. 端口说明

### 输入

- **输入 0**（ImageData）：保留端口，当前未使用。

### 输出

- **输出 0**（ImageData）：当前摄像头帧。

## 3. 界面说明

- **设备下拉框**：列出可用摄像头，选择后自动开始采集。
- 无可用设备时显示提示文字。

## 4. 使用说明

1. 添加节点后在下拉框中选择摄像头。
2. 将输出接到 Scale Image、NDI Out 等下游节点。
3. 更换设备时在下拉框中重新选择即可。

## 5. 示例

USB 摄像头 → Camera → Scale Image（1280×720）→ Image Compare，用于现场画面监测。
