# PLC-ModBus

## 1. 节点说明

通过 **Modbus TCP** 读写 PLC 保持寄存器，实现最多 32 路 **Z**（控制输出）与 **S**（状态反馈）。支持 **Beckhoff**、**Siemens** 两种地址映射；Z 为脉冲式写入（约 500 ms 后自动复位），S 由定时轮询更新。

## 2. 端口说明

### 输入

| 端口 | 类型 | 说明 |
|------|------|------|
| Z 0 … Z n | VariableData | 布尔，触发对应控制位（默认 8 个端口，可编辑至 32 路逻辑） |

### 输出

| 端口 | 类型 | 说明 |
|------|------|------|
| S 0 … S n | VariableData | 布尔，对应当前状态位反馈 |

## 3. 界面说明

- **主机 / 端口 / 从站 ID**：Modbus TCP 连接（默认 `127.0.0.1:502`）。
- **PLC 类型**：Beckhoff 或 Siemens，切换后更新 Z/S 标签与寄存器基址。
- **Z 0～31**：控制按钮（切换触发）。
- **S 0～31**：只读状态指示。
- **连接状态**：是否在线。

外部控制：`/host`、`/port`、`/serverId`、`/plcType`、`/Z0`～`/Z31`。

## 4. 使用说明

1. 选择 PLC 类型，填写 IP 与端口，连接成功后可操作 Z。
2. 将 Z 输入设为 `true` 发出脉冲；界面按钮同理。
3. 从 S 输出读取 PLC 状态，用于联锁或显示。
4. 界面地址为 1 基展示，内部会按映射转换为协议地址。

## 5. 示例

S 5 为 `true`（门已关）→ 逻辑允许 → Z 2 写入 `true` 启动幕帘；S 接条件节点，Z 接本节点输入。
